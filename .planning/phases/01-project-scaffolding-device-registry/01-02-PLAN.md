    Define `ChecksumType` enum:
    ```rust
    #[derive(Debug, Clone, Copy, PartialEq, Eq, Default, Serialize, Deserialize)]
    pub enum ChecksumType {
        #[default]
        Bit7,
        Bit8,
        None,
    }
    ```

    Implement three functions exactly matching the reference:

    `pub fn calculate_checksum(data: &[u8], checksum_type: ChecksumType) -> u8`:
    - Bit7: sum bytes 0..7 of data (first 7 bytes), return `(255 - (sum & 0xFF)) as u8`
    - Bit8: sum bytes 0..8 of data (first 8 bytes), return `(255 - (sum & 0xFF)) as u8`
    - None: return 0
    Note: `data.iter().take(N)` where N=7 for Bit7, N=8 for Bit8. Sum as u32 to avoid overflow.

    `pub fn apply_checksum(data: &mut [u8], checksum_type: ChecksumType)`:
    - Bit7: if data.len() >= 8, set `data[7] = calculate_checksum(data, checksum_type)`
    - Bit8: if data.len() >= 9, set `data[8] = calculate_checksum(data, checksum_type)`
    - None: no-op

    `pub fn build_command(cmd: u8, data: &[u8], checksum_type: ChecksumType) -> Vec<u8>`:
    - Allocate `vec![0u8; crate::hid::REPORT_SIZE]` (65 bytes)
    - `buf[0] = 0` (USB report ID)
    - `buf[1] = cmd`
    - Copy `data` into `buf[2..2+len]` where `len = min(data.len(), REPORT_SIZE - 2)`
    - `apply_checksum(&mut buf[1..], checksum_type)` -- CRITICAL: checksum starts at buf[1] (the cmd byte), NOT buf[0] (report ID)
    - Return buf

    `pub fn build_ble_command(cmd: u8, data: &[u8], checksum_type: ChecksumType) -> Vec<u8>`:
    - Allocate `vec![0u8; crate::ble::REPORT_SIZE]` (66 bytes)
    - `buf[0] = crate::ble::VENDOR_REPORT_ID` (0x06)
    - `buf[1] = crate::ble::CMDRESP_MARKER` (0x55)
    - `buf[2] = cmd`
    - Copy `data` into `buf[3..3+len]` where `len = min(data.len(), ble::REPORT_SIZE - 3)`
    - `apply_checksum(&mut buf[2..], checksum_type)` -- checksum starts at buf[2] (the cmd byte)
    - Return buf
