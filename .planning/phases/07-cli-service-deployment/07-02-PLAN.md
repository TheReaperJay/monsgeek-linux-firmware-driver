  <action>
Create `deploy/systemd/monsgeek-driver.service`:
- `[Unit]`:
  - `Description=MonsGeek Driver Bridge`
  - `After=network-online.target`
  - `Wants=network-online.target`
- `[Service]`:
  - `Type=simple`
  - `ExecStart=/usr/bin/monsgeek-driver`
  - `Restart=on-failure`
  - `RestartSec=2`
  - `Environment=RUST_LOG=info`
  - `NoNewPrivileges=true`
- `[Install]`:
  - `WantedBy=multi-user.target`

Create `deploy/systemd/monsgeek-inputd.service`:
- `[Unit]`:
  - `Description=MonsGeek Input Daemon`
  - `After=monsgeek-driver.service`
  - `Requires=monsgeek-driver.service`
- `[Service]`:
  - `Type=simple`
  - `ExecStart=/usr/bin/monsgeek-inputd --debounce-ms 15`
  - `Restart=on-failure`
  - `RestartSec=2`
  - `NoNewPrivileges=true`
- `[Install]`:
  - `WantedBy=multi-user.target`

Do not introduce shell wrappers in `ExecStart`; keep direct binary invocation for predictable systemd supervision.
  </action>
